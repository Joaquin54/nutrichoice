# NutriChoice

*Eat better. Feel better. Choose smarter.*

NutriChoice is a web application built to make healthy eating feel less like a chore and more like a natural part of your day. You can discover and save recipes, log your meals, track your macros, and plan your week — all in one place, tailored to your own dietary goals.

## The thought behind it

We kept running into the same problem: good nutrition information existed, but it was scattered. You'd check one app for calories, another for recipes, and still end up with a spreadsheet to pull it together. NutriChoice was born from the idea that choosing what to eat should feel effortless and informed — not like a research project. We wanted to build something that actually fits into how people eat, not how apps think they should.

## Running locally

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

**Backend**
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 0.0.0.0:8000
```
